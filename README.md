## 使用介绍

1. step-1

```
git clone https://github.com/hysfwjr/vimhome.git vimhome

cd vimhome

#if windows
cp _vimrc ../_vimrc
#if linux
cp _vimrc ../.vimrc

git clone https://github.com/gmarik/vundle.git
```

2. step-2 添加环境变量
export VIMHOME=/path/to/vimhome

3. step-3 执行BundleInstall安装插件
`:BundleInstall`
